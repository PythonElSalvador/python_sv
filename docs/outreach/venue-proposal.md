# Venue Proposal — Template

Reusable template for reaching out to universities or venues to host Python SV events.

## Template

```
Hola {contact_name}, buenas tardes!

Espero que esté bien. Como le comenté, ya estamos organizando el próximo evento de Python SV y quería compartirle los detalles:

Fecha: {event_date}
Horario: Por definir — nos adaptamos a la disponibilidad del espacio
Formato: Podemos adaptarnos a lo que más les convenga. Algunas opciones:
- Meetup (2-3 horas): 2-3 charlas cortas + networking
- Workshop (3-4 horas): sesión práctica donde los asistentes programan en vivo
- Combinación de ambos: charlas + un taller hands-on
- PyDay (día completo, 6-8 horas): varias charlas, talleres, y networking — un evento más grande

Nos encantaría que la universidad le consulte a sus estudiantes o interesados qué formato y temas les gustaría ver, y nosotros buscamos speakers para eso. Queremos que esto sea colaborativo para que todos se beneficien.
Asistentes estimados: Mínimo 40 personas, pero queremos atraer la mayor cantidad posible — eso lo podemos coordinar con ustedes según la capacidad del espacio
Costo: Gratuito y abierto al público

También si hay estudiantes que quieran dar una charla corta o un demo, con mucho gusto les damos ese espacio. La idea es que se sienta como un evento de la comunidad y de la universidad, no solo nuestro.

¿Habría posibilidad de usar un salón o auditorio en {university_short} para esa fecha? Necesitaríamos un espacio con proyector y capacidad para unas 40-50 personas. También estamos planeando ofrecer comida a los asistentes — ¿sería posible coordinar eso dentro del espacio?

Con gusto los mencionamos como sede oficial en toda la promoción del evento (redes sociales, sitio web pythonsv.com, y durante el evento).

Quedo atento, muchas gracias!
Kevin
```

### Placeholders

| Placeholder            | Description                          | Example                       |
|------------------------|--------------------------------------|-------------------------------|
| `{contact_name}`       | First name of the contact            | Miguel                        |
| `{event_date}`         | Event date in Spanish                | Sábado 2 de mayo, 2026       |
| `{university_short}`   | Short name of the university         | la UEES                       |

**Note:** The template assumes a first contact or cold outreach. For existing WhatsApp threads, drop the "Como le comenté" intro — jump straight to the event details.
